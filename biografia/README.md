# 🌐 Israias Marques - Personal Portfolio Website

## Sobre o Projeto

Este projeto consiste em uma **página web estática de apresentação pessoal**, desenvolvi utilizando **HTML5 e CSS3**, com o objetivo de centralizar informações sobre a trajetória, projetos e minhas iniciativas **Israias Alves Marques**.

A aplicação funciona como um **portfólio digital**, apresentando de forma simples e organizada informações profissionais, projetos desenvolvidos e links relevantes.

O projeto foi estruturado utilizando apenas tecnologias fundamentais da web, permitindo fácil manutenção, aprendizado.

---

## 🎯 Objetivo

O objetivo deste projeto é:

* Servir como **página pessoal online**
* Apresentar a **biografia e trajetória profissional**
* Divulgar **projetos e aplicações desenvolvidas**
* Demonstrar conhecimentos básicos em **desenvolvimento web**
* Disponibilizar um **portfólio digital acessível pela web**

---

## 💻 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando tecnologias fundamentais da web:

* **HTML5** – Estrutura da página
* **CSS3** – Estilização e layout

Por se tratar de um projeto **estático**, não são necessárias dependências externas, frameworks ou bibliotecas adicionais.
